Paas

Poucas pessoas de negócios irão interagir com um PaaS, sigla para Plataforma como Serviço, já que é uma tecnologia voltada, principalmente, para desenvolvedores e profissionais de criação de sistemas. As soluções de PaaS são usadas, na maioria das vezes, para ter um ambiente de desenvolvimento, teste e gerenciamento de aplicativos, sendo a escolha perfeita para empresas de desenvolvimento de software. Graças às soluções de PaaS, os desenvolvedores de software podem criar novos sistemas sem a necessidade de toda a infraestrutura comum que envolve componentes como: Servidores; Bancos de dados; Sistemas operacionais; Ferramentas de desenvolvimento.
